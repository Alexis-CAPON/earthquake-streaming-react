## Quick start

You need to start 2 terminals, one in the backend folder and the other one in the frontend.

```
cd earthquake-streaming-react
```

First terminal:
```
cd frontend-web
yarn install && yarn start
```

Second terminal:
```
cd backend-express
yarn install && nodemon server.js
```
